Features: Cool features of RobBot

Features

  • RobBot detects online harassment and cyberbullying through tweets and gets set to work by retweeting and saving the user into a database.
  • This project is the winner of the #HackHarassment prize at HackRPI'16!

More information:

  • RobBot is written in Javascript and uses node.js as a Javascript runtime to run Download node.js here https://nodejs.org/en/
  • Uses Twit open source API. Twit expands upon the functionality of Twitterâ€™s API, allowing for the simple access of Twitter through Javascript code. https://www.npmjs.com/package/twit
  • Download source code. Change the fields in the config.js file to gain access to your twitter account. https://dev.twitter.com/overview/api
  • Run Node.js runtime environment on your window console (ex. node robBot.js) to start the bot.
